# Number-Guessing-Game
An interactive game where you guess a randomly generated number, with a catch where you mustn't lose all your lives in the process. Scripted in Python with little-to-no coding experience.

## Notable things
* Multiple round system
* Difficulty scaling (# of lives based on difficulty)
* Basic error-handling

## Instructions
* Ensure Python is installed on your computer.
* Download/clone this repo
* Navigate to the project folder
* Run the program
* Enjoy!
